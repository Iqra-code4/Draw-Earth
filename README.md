# Draw-Earth
Draw-Earth is a platform where creators can record their skills and achievements from hackathons, coding to sketching and painting. Users can build rich profiles, upload their work, and show their talents on one space.  The platform also allows users to form teams for hackathons, and creative collaborations by connecting with like-minded people.
